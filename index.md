# AI Coding 项目介绍

这里是一个面向 AI 编码场景的项目入口页面，聚焦“更快、更准、更稳”的开发体验。你可以在此了解本项目的目标、能力边界以及核心优势。

## 能力亮点

- 智能补全：基于上下文的代码补全与片段推荐。
- 自动审查：发现潜在 Bug 与风格问题，提升代码质量。
- 测试生成：自动生成基础测试用例，降低维护成本。
- 重构建议：提供可操作的重构路径与风险提示。
